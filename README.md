How to use: Setup guide

note for `Windows`: Use `git bash` terminal if you are on windows.

In your git bash terminal, copy and run:
```bash
iwr https://raw.githubusercontent.com/fersonull/rn-dependency-installer/main/setup.sh | bash
```
For (Linux/Mac) just copy and run:
```bash
curl -fsSL https://raw.githubusercontent.com/fersonull/rn-dependency-installer/main/setup.sh | bash
